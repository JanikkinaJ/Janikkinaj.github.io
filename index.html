Vidconvert Mass batch conversion using ffmpeg. Converts video files into Mp4 and then sorts them

Compatibility

Right now vidconvert only supports Linux but macOS and Windows are on the way

Installation

clone the repository: "git clone https://github.com/JanikkinaJ/vidconvert.git" then "cd vidconvert" then "sudo make setup" to install onto system

Usage

after install just move inside the directory where you want to convert files to mp4, then type vidconvert.py
